# Healthcare - IoT Deception Honeypot Network

## Description
A Healthcare IoT Deception Honeypot Network built using Cowrie, Docker, Python, and Splunk to simulate vulnerable medical IoT devices, capture attacker activities, and analyze threats through security monitoring and visualization.

# Features

* Healthcare IoT device simulation
* Cowrie honeypot deployment
* SSH/Telnet attack monitoring
* Attacker activity logging
* Malware file capture
* Docker-based isolation
* Python log analysis
* Splunk dashboard visualization
* Threat intelligence generation
* Attack source IP tracking

# Technologies Used

* Ubuntu 22.04
* Cowrie Honeypot
* Docker
* Python 3
* Splunk Enterprise
* Git & GitHub
* SSH/Telnet Protocols

# Conclusion

This project demonstrates the development of a healthcare-focused Cowrie SSH honeypot for detecting and analyzing cyber threats targeting medical environments. A realistic healthcare device was simulated using a customized filesystem, while Splunk was integrated to collect, monitor, and visualize attacker activity through interactive dashboards. Python scripts were developed to automate IoC extraction, generate threat intelligence reports, and perform IP geolocation analysis. Overall, the project provides a practical approach to understanding attacker behavior, improving threat visibility, and strengthening cybersecurity monitoring for healthcare systems.
